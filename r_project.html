My Resume Website

This website serves as my personal portfolio and resume. It is a full-stack web development project where I have built an interactive and responsive webpage to showcase my skills, projects, and educational background.

The project was developed using technologies such as HTML, CSS, JavaScript, and a solid understanding of full-stack principles. It was designed to highlight my personal information, skills, and projects while ensuring that the page remains user-friendly, modern, and mobile-responsive.

I implemented various features such as a **Go Home** button for easy navigation, sections like **About Me**, **Education**, and **Languages**, and made sure the website adapts smoothly to different screen sizes.

This project has been a great opportunity for me to apply my knowledge of front-end and back-end technologies, while refining my web development and design skills.
